UI Clone

## Tech Stack

**Client:** React JS
**Sever:** Node Js

## Run Locally

Clone the project

```bash
  git clone https://github.com/raghav-03/instaacoders.git
```

Go to the project directory

```bash
  cd INSTACODERS
```

## FrontEnd

Hosted Link - https://instaacoders.vercel.app/

Install dependencies

```bash
  cd frontend/
  npm install
```

Start the server

```bash
  npm run start
```
